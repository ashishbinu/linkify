# Linkify

A Mozilla Thunderbird extension to automatically convert plain text urls to clickable links.

# Requirements

* `git`
* `pnpm`

# Developer Installation 

* `git clone https://github.com/ashishbinu/linkify`
* `pnpm install`
* `pnpm dev`
